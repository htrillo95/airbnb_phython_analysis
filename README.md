# Airbnb Paris Analysis

A quick analysis of Airbnb listings in Paris using Python and pandas.

## What's in here

- `airbnb_analysis.ipynb` - Jupyter notebook with the analysis
- `Listings.csv` - Airbnb listings data
- `Reviews.csv` - Reviews data
- Data dictionaries for reference

## Getting started

Just open the notebook and run the cells. The data files are included (tracked with Git LFS since they're large).

## Requirements

- Python 3
- pandas
- Jupyter notebook

Install dependencies:
```bash
pip install pandas jupyter
```
